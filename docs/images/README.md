# Images

This directory contains recruiter-facing visual assets for BetCopilot AI.

## Architecture Assets

- `betcopilot-architecture.mmd` - Mermaid source for GitHub rendering
- `betcopilot-architecture.svg` - polished standalone diagram for docs, sharing, and screenshots

## Screenshot Placeholders

Recommended screenshot filenames:

- `demo-hero.png`
- `demo-results.png`
- `demo-eval.png`

## Optional PNG Export

Use [`../../scripts/export_diagram.ps1`](../../scripts/export_diagram.ps1) to export the Mermaid source to PNG or SVG with Mermaid CLI once dependencies are available locally.
